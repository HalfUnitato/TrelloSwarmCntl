# TelloSwarmCntl


This repository provides the code and instructions to set-up the control of a single [Tello drone](https://www.ryzerobotics.com/), as well as a swarm control via a command file.

## Prerequisites

´pipenv´
´python >= 3´

## Setup

clone the repository

´´´

# In project root-dir
pipenv shell

pipenv install
´´´
